# Библиотека zaboy 
##Настройка
- Переменную окружения `APP_ENV` установите в `'dev'` или в `'prod'`.  
- Создайте MySQL BD.
- Создайте файл `config\autoload\db.local.php` по образцу `db.global.php`
- Запустите скрипт `src\Installer.php`.
